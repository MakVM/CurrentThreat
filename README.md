# CurrentThreat - AI Cyber Threat Intelligence Dashboard
An AI-powered product made from the [Product Kit Template](https://github.com/1712n/product-kit-template) using [CPW API](https://rapidapi.com/CPWatch/api/cpw-tracker).

CurrentThreat is an AI-powered cyber threat intelligence platform that automatically aggregates, enriches, and visualizes daily and weekly cybersecurity events. Using real-time event data, the platform generates a daily threat score, a weekly intelligence brief, and interactive visualizations to help security analysts, researchers, and organizations monitor the cyber landscape.

## Use Case

CurrentThreat helps users quickly understand the **current cybersecurity threat landscape** without manually tracking multiple sources.  

Designed for:

- Security analysts who need **daily briefings** of critical events.
- Researchers tracking **trends and predictions** in cyberattacks.
- Organizations building **alert systems** and monitoring dashboards.
- Enthusiasts or educators who want to **visualize cybersecurity intelligence** interactively.

The project leverages AI models to **summarize, classify, and score events**, providing actionable insights from raw data.

---

## Features

### Daily Threat Monitoring
- Automatically fetches and enriches cybersecurity events from multiple sources.
- Assigns **severity levels** (High, Medium, Low) with color-coded visual cues.
- Generates a **daily threat score** (0–10) using AI analysis.
- Provides a **“current threat” codename** summarizing the day’s cyber landscape.

### 7-Day Rolling Intelligence
- Aggregates **past week’s events** into trends, predictions, and executive briefings.
- AI-generated weekly summaries provide actionable insights.
- Interactive **charts** show event frequency and threat classification distribution.

### Interactive Dashboard
- **Responsive design** with modern cyber-themed visuals.
- Hover effects, glow animations, and dynamic charts.
- Clickable **source links** for each event.
- Social media sharing for individual events and intelligence summaries.

### Automation & AI Integration
- Fully automated **daily updates** via GitHub Actions.
- AI-powered summarization, classification, and scoring.
- Leverages GitHub Models for **prompt-driven analysis**.
- Optional social media posting (Twitter, Facebook) of daily events.
